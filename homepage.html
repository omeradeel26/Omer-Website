Checkers
×
Description
This project is inspired by the board game Americain Checkers. It was coded in Python and created with the GUI Processing. It took approximately 25 hours to code from scratch and was created for my grade 11 ICS final. The goal of the game is to see which player can get all of the other players checkers first. The skills required to build this project include a knowledge of loops to predict open spots with the jumping mechanim, knowledge of variables to control different factors of the game such as player turn or the number of checkers left and knowledge of UI to create different features such as a homescreen, gamescreen, instruction screen, and endgame screen. Check out the Github Repo linked.
